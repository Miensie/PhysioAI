Guide de déploiement

En ligne en moins de 30 minutes

GitHub Pages pour le frontend (gratuit), Render.com pour le backend Python (gratuit).

🌐 GitHub Pages — Frontend
01
Créer le dépôtSur github.com/new → nommer physioai-lab → Public
02
Pousser les fichiers 
git init && git add .
git commit -m "feat: PhysioAI Lab v2"
git remote add origin https://github.com/
  VOTRE-USERNAME/physioai-lab.git
git push -u origin main
03
Activer GitHub PagesSettings → Pages → Branch: main / root → Save
→ Disponible sur https://USERNAME.github.io/physioai-lab
04
Configurer les secrets CISettings → Secrets → Actions → New secret :
BACKEND_URL = URL de votre backend Render
🚀 Render.com — Backend Python
01
Créer un compte gratuitSur render.com → Dashboard → New Web Service
02
Connecter le dépôt GitHubSélectionner votre repo physioai-lab → Root Directory : backend
03
Configurer le service 
Build Command :
pip install -r requirements.txt

Start Command :
uvicorn main:app --host 0.0.0.0 --port $PORT
04
Variables d'environnement 
Environment Variables :
ENVIRONMENT   = production
FRONTEND_URL  = https://USERNAME.github.io/physioai-lab
05
Récupérer le Deploy HookSettings → Deploy Hooks → Copier l'URL → Ajouter en secret GitHub : RENDER_DEPLOY_HOOK
⊞ Chargement dans Excel
01
Mettre à jour manifest.xmlRemplacer localhost:3000 par votre URL GitHub Pages dans manifest.xml
02
Excel DesktopInsérer → Compléments → Mes compléments → Télécharger → Sélectionner manifest.xml
03
Excel OnlineInsérer → Compléments Office → Charger mon complément → manifest.xml
04
Configurer l'URL backendDans l'Add-in : champ URL en haut → saisir l'URL Render → le statut devient vert ●
🖥️ Alternative — VPS avec Docker
01
Installer Docker sur le VPS 
curl -fsSL https://get.docker.com | sh
02
Build et lancement 
docker build -t physioai-backend .
docker run -d \
  -p 8000:8000 \
  -e ENVIRONMENT=production \
  -e FRONTEND_URL=https://... \
  --name physioai \
  physioai-backend
03
HTTPS avec Nginx + Let's Encrypt 
apt install nginx certbot
certbot --nginx -d api.votre-domaine.com
